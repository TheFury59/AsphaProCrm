# Assets

Placeholders a remplacer par les vrais visuels Aspha :

- `icon.png` — 1024x1024, app icon (utilise iOS + fallback Android)
- `adaptive-icon.png` — 1024x1024, foreground Android (zone safe 432x432 au centre)
- `splash.png` — 1284x2778, splash screen (fond vert #10b981)
- `favicon.png` — 48x48, favicon web

En attendant les visuels finaux, l'app utilisera les defaults Expo (carre blanc).
Tu peux ignorer les warnings au demarrage tant que les fichiers ne sont pas la.
